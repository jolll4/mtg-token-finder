## Usage

Download Oracle cards as oracle-cards.json from https://scryfall.com/docs/api/bulk-data. Put your decklist to a text-file and run.

```bash
python ./tokens.py -p "decklist.txt"
```

Format options (WIP):

`withCount` (default)

```bash
1 Raise the Alarm
```

justName

```bash
Raise the Alarm
```
